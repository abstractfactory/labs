### Flask Bootstrap

An implementation of the tutorial by [Brad Hussey][tut] adapted for Flask and Jinja2.

### How to Get Started

1. Download this experiment

2. Install dependencies

 ```bash
$ pip install flask
```

3. Run the app

 ```bash
$ python app.py
```

4. Browse to app

 > [http://localhost:5000](http://localhost:5000)
